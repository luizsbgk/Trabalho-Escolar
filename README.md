# 🛡️ Como Hackers Invadem Sistemas

Este é um projeto desenvolvido para fins educacionais (Trabalho Escolar) que explora e explica as principais metodologias utilizadas em ataques cibernéticos.

📋 Sobre o Projeto
O objetivo deste site é conscientizar sobre segurança digital, demonstrando de forma visual e estruturada como vulnerabilidades são exploradas no mundo real.

🚀 Tecnologias Utilizadas
HTML: Estruturação do conteúdo e textos explicativos.
CSS: Estilização visual (design focado em uma temática "Hacker/Terminal").

📂 O que o site aborda?
- [ ] Engenharia Social (Phishing).
- [ ] Ataques de Força Bruta.
- [ ] Injeção de SQL.
- [ ] Medidas de prevenção e boas práticas.

💻 Como visualizar
Como o projeto utiliza apenas tecnologias front-end básicas, você pode:
1. Clonar o repositório.
2. Abrir o arquivo `index.html` em qualquer navegador.
